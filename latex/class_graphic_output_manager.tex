\subsection{Detailed Description}
An output manager that manages a Window and a Graphic User Interface This class owns \mbox{\hyperlink{class_game_sprite}{Game\+Sprite}} objects, \mbox{\hyperlink{class_sprite_manager}{Sprite\+Manager(s)}}, S\+DL textures and a S\+DL window and renderer. It is dependent on \mbox{\hyperlink{class_sprite_manager}{Sprite\+Manager}} to execute many of the tasks related to dynamic sprites.
